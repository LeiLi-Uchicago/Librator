Installation Instructions

Install Librator

  • Download DMG file from Here
  • Double click the DMG file and then drag and drop "Librator" to "Applications" folder

Install Dependencies

This software require:
Muscle (Required)
Clustal Omega (Required)
RAxML (Recommended, for phylogenetic tree)
PyMOL (Recommended, for 3D visualization)
FigTree (Recommended, for phylogenetic tree visualization)

We already included Muscle, Clustal Omega and RAxML in Librator. If those pre-complied softwares don't work on your computer, you can download or complie from offical website and determine the path for those tools in Setting
Install PyMOL from the following way:
  • Download from offcial website PyMOL (license required after 30 days free trail)
  • Install from anaconda

    • conda install -c schrodinger pymol

  • Install from homebrew

    • brew install brewsci/bio/pymol

  • Install from MacPorts

    • sudo port install pymol

  • Install from source

    • python setup.py --osx-frameworks install
